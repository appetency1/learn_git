<!-- ========== 二、段落与换行 ========== -->
<!--
【重要坑点】
1. 直接按一次回车 = 空格，不会换行
2. 空行（两次回车）= 分段，间距大
3. 行尾两个空格 + 回车 = 软换行，间距小
4. <br> = 强制换行，最直观
-->

这是第一段。
两段之间必须有空行，否则连在一行。

这是第二段，和上面有空行，所以分段了。

行尾加两个空格  
才能实现软换行（诗歌、地址常用）。

也可以用 HTML 标签<br>直接换行，不容易忘。

<!-- ========== 三、文字样式 ========== -->
<!--
【注意】
1. 标记符号和文字之间不能有空格：** 文字 ** 无效
2. 粗体用 ** 或 __，斜体用 * 或 _
3. 删除线用 ~~，高亮用 ==（部分编辑器不支持）
4. 下划线 Markdown 没有原生语法，用 <u> HTML 标签
-->

**粗体** 或 __粗体__
*斜体* 或 _斜体_
***粗斜体*** 或 ___粗斜体___
~~删除线~~
==高亮==（Typora 支持，VS Code 需插件）
<u>下划线</u>（HTML 标签）
`行内代码`（反引号包裹）
